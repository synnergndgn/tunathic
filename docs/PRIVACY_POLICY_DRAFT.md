# Tunathic Privacy Policy Draft

**Status:** Product draft for the current Phase 1C application. This is not a final store-publishing policy and contains no invented legal contact details.

Tunathic – Guitar Toolkit is published by GUNDEV. The current application is designed to operate offline and keep its practice data on the user’s device.

## Current data behavior

- BPM Tap sessions exist only in application memory. Tap timestamps and estimates are not saved as session history or uploaded.
- Theme, language, haptic-feedback, and Metronome preferences are stored locally on the device.
- The application does not request microphone permission.
- The application does not record, process, or upload microphone audio.
- No account is required.
- The application contains no advertising or analytics SDK.
- Tunathic has no application backend in this release.
- The application sends no app data to GUNDEV servers.

Platform services and the app-distribution provider may process technical information under their own policies. Tunathic does not add its own remote collection in the current release.

## Future changes

This draft must be reviewed and updated before releasing any feature involving microphone access, audio recording, advertising, analytics, accounts, cloud synchronization, purchases, or a Tunathic backend. Those features must not inherit this draft without an accurate new data-flow review.

## Publication status

This document describes the current implementation but does not claim Play Store readiness. Final publication requires legal review, effective-date and policy-hosting decisions, and any store-required disclosures.
